Analise de Dados

Dados abertos: são aqueles que qualquer pessoa pode acessá-los e utilizá-los
Big Data: é uma enorme quantidadde de dados desestruturados, grande demais para ser manipulada por infraestruturas tradicionais Apaache Hadoop: framework usado para realizar o processamento e análise de dados estruturados e não estruturados Dados estruturados: são organizados em tabelas, com estrutura precviamente planejada Dados não estruturados: apresenta estrutura dinânima. Alguns exemplos são Textos, arquivos, documentos, imagens, vídeos, áudios, redes sociais e outros dados compartilhados na web.
