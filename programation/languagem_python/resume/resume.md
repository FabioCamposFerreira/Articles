###\section{Como pegar o texto digitado pelo terminal?}

```python

```
  
###  \section{Como imprir textos coloridos?}
  
```python
text = "Text red"
print("\033[91m {}\033[00m".format(text))
# Text red
text = "Text green"
print("\033[92m {}\033[00m".format(text))
# Text green
```
